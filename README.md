# ATM-Management-System-Java
Java-based ATM Management System with GUI, file handling, and secure user authentication. Supports balance inquiry, deposit, withdrawal, PIN change, and money transfer.
This project is a Java-based ATM Management System developed using Swing for GUI and file handling for data persistence.

The system simulates real-world ATM operations such as user authentication, balance checking, deposit, withdrawal, money transfer, and PIN change. User data is securely stored and managed using a text file, ensuring that all transactions are saved.

This project demonstrates the use of object-oriented programming concepts, GUI development, and file handling in Java, making it a practical implementation of a real-world banking system.
